About This Project

Project Overview

This project demonstrates the complete DevOps workflow including version control with Git, remote repository management with GitHub, and containerization with Docker.

Assignment Objectives

  • Set up and manage a local Git repository
  • Demonstrate various Git commands (init, status, add, commit, log, branch, switch)
  • Integrate with GitHub for remote repository management
  • Create a custom Docker image for the application
  • Push the Docker image to Docker Hub
  • Run and manage Docker containers

Technologies

Version Control: Git, GitHub

Containerization: Docker, Docker Hub

Web Framework: Flask (Python)

Frontend: HTML5, CSS3
